# ¿Cuál es mi IP?

Este proyecto es una herramienta minimalista para ver tu dirección IP pública directamente desde el navegador.

### 🔧 Tecnologías utilizadas
- HTML + CSS + JavaScript
- API de ipify.org
- Hosting en GitHub Pages
